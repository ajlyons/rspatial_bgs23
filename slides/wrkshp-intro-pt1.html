Workshop Goals

Get better at:

  • Working with R and RStudio
  • Using the most common data classes and packages for spatial data
  • Importing spatial data from files and online sources
  • Fundamental spatial data processing and geoprocessing operations
  • Basic plotting
  • Finding resources to continue learning on your own


Wind up a little higher on the learning curve!

Strategy: Copy and Adapt

foundational concepts
+ code recipes
+ working examples
+ practice
+ a good search engine

functional pRoficiency!

Advice for Beginners

  1. Try to get working code
  2. Focus on adapting code rather than starting from scratch
  3. Pay attention to syntax
  4. Use meaningful variable names
  5. Ask for help for cryptic error messages
  6. Comment your code liberally
